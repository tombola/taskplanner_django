Task Templates

Help

A task template is a repeatable group of tasks that can be created all together.

For example, create a task template for squashes, and define the 'tasks' as 'sow', 'harvest', and 'process'.

When you then click 'create todos' from that template it will present a form for SKU, crop, variety - submitting that form will create todoist tasks for 'sow', 'harvest', and 'process', all grouped under a parent task.
